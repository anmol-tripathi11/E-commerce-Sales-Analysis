🛒 E-commerce Sales Analysis (Excel Dashboard)

An Excel-based analytics dashboard designed to analyze e-commerce sales data and generate meaningful business insights. This project focuses on identifying sales trends, customer behavior, and performance metrics using interactive visualizations.

📌 Project Overview

This dashboard helps users:

Understand overall sales performance
Analyze customer purchasing patterns
Track key business metrics in real-time
Make data-driven decisions easily
✨ Features
📊 Interactive dashboard with slicers
📈 Key Performance Indicators (KPIs)
🔄 Dynamic charts and reports
📦 Sales and profit analysis
👤 Customer behavior insights
⚡ Easy-to-use and visually clean interface
🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning & Processing
📊 KPIs Included
Total Sales
Total Orders
Profit
Quantity Sold
Profit Margin
